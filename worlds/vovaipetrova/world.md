# VovaIPetrova — Семантический мир

## Философия

Это досмысловой мирок — набросок UI, спектакль, туториал, пластинка на проигрывателе.

Мир показывает, как работает MeaningEngine, но не является его частью.

## Структура

```
Universe (root)
├── Characters (hub)
│   └── Вова (character)
├── Domains (hub)
│   └── Art, Music, ... (domain)
└── Cryptocosm (cryptocosm)
    └── Рефлексивная часть
```

## Принципы

1. **UI принадлежит миру** — не Engine
2. **Schema принадлежит миру** — не Engine
3. **Seed принадлежит миру** — не Engine
4. **Engine работает через абстракции** — не знает конкретных типов
