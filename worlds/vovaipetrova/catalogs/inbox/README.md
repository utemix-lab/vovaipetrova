# Inbox — приёмник LLM-ответов

## Как использовать

1. Получи ответ от LLM (ChatGPT/Claude)
2. Сохрани в файл `inbox/batch-XX.json`
3. Скажи мне: "Обработай batch-01"
4. Я валидирую и перенесу в `vst-plugins.json`

## Формат файла

```json
{
  "batch": 1,
  "source": "chatgpt-4",
  "date": "2026-02-18",
  "instruments": [
    { ... },
    { ... }
  ]
}
```

## Экономия токенов

- Ты не вставляешь JSON в чат → экономия ~80% токенов
- Я читаю файл напрямую → 0 токенов на передачу
- Валидация автоматическая → меньше итераций

## Workflow

```
LLM → batch-XX.json → validator → vst-plugins.json
         ↑                              ↓
    ты сохраняешь              я коммичу
```
