# UI Index (contracts only)

## Files
- `layout/visitor.layout.json`  
  Layout panels + responsive rules.
- `bindings/visitor.bindings.json`  
  Maps element IDs to assets/content.
- `interaction/visitor.interaction.json`  
  Event rules and global bindings.
- `widgets/domains.json`  
  Visual anchors for Story panel.

## Rules
- `UI_RULES.md` is the single source of UI mechanics.
- `UI_CONTRACT_STYLE.md` defines JSON style and top-level fields.
