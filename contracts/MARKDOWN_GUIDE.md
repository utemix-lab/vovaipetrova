# Markdown Guide для contracts

Godot sandbox поддерживает подмножество Markdown, которое конвертируется в BBCode.

## Поддерживаемый синтаксис

### Заголовки

```markdown
# Заголовок 1 (26px)
## Заголовок 2 (22px)
### Заголовок 3 (18px)
```

### Форматирование текста

```markdown
**жирный** или __жирный__
*курсив* или _курсив_
~~зачёркнутый~~
`код`
```

### Ссылки

```markdown
[текст ссылки](https://example.com)
```

Ссылки кликабельны и открывают браузер.

### Теги

```markdown
#ecosystem #demo #visitor
```

Теги подсвечиваются фиолетовым и кликабельны.

### Списки

```markdown
- Элемент 1
- Элемент 2
* Альтернативный синтаксис

1. Нумерованный
2. Список
```

### Цитаты

```markdown
> Это цитата
> На несколько строк
```

### Горизонтальная линия

```markdown
---
```

## Пример полного текста

```markdown
# Введение

Это **важный** текст с *акцентом*.

## Особенности

- Поддержка #тегов
- Кликабельные [ссылки](https://example.com)
- `Код` в строке

> Цитата мудреца

---

### Заключение

~~Неактуально~~ → Актуально!
```

## Размещение текстов

```
contracts/public/texts/
├── story/      ← Нарративные тексты
├── system/     ← Технические описания
└── service/    ← Инструкции и действия
```

## Использование в route.json

Вместо inline текста можно ссылаться на файл:

```json
{
  "story": {
    "text_file": "texts/story/welcome.md",
    "refs": [...]
  }
}
```

*(Эта функция планируется)*
