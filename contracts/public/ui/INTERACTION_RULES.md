# dream-graph: Реализация согласованности проекций

> Этот документ описывает **КАК** dream-graph реализует согласованность проекций.  
> **ПОЧЕМУ** это правило существует — см. ADR-004.  
> **ЧТО** это правило — см. policy-projection-coherence в System Graph.

---

## Реализация синхронизации

При взаимодействии с любой проекцией узла:

| Проекция | Реакция |
|----------|---------|
| Узел на графе | `--color-active` (жёлтый) |
| Рёбра узла | `--color-active` (жёлтый) |
| Виджет в панели | Жёлтая рамка + цветная иконка |

## Цветовая палитра

| Состояние | CSS Variable | Hex |
|-----------|--------------|-----|
| Неактивно | — | grayscale |
| Hover | `--color-semi-active` | #22d3ee |
| Активно | `--color-active` | #fbbf24 |

## Детали реализации виджетов

- Только внутренняя подсветка (без box-shadow)
- Иконки: grayscale → color при hover
- Сдвиг: `translateY(-2px)`
