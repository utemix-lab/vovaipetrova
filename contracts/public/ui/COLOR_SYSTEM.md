# UI Color System

## Semantic Colors (состояния интерфейса)

| Состояние | Цвет | Hex | Применение |
|-----------|------|-----|------------|
| **Semi-active** | Голубой/Cyan | `#22d3ee` | Hover, связи, полуактивность |
| **Active** | Жёлтый/Amber | `#fbbf24` | Выбор, фокус, активность |
| **Neutral** | Серый | `#8a8a8f` | Неактивные элементы, текст |

## Правила

1. **Граф** — единственный цветной элемент (узлы по типам)
2. **UI** — monochrome (серые оттенки)
3. **Состояния** — голубой (hover) → жёлтый (active)

## CSS Variables

```css
/* Полуактивность (hover, связи) */
--color-semi-active: #22d3ee;
--color-semi-active-soft: rgba(34, 211, 238, 0.15);

/* Активность (выбор, фокус) */
--color-active: #fbbf24;
--color-active-soft: rgba(251, 191, 36, 0.2);
--color-active-glow: rgba(251, 191, 36, 0.25);
```

## Принцип

При смене палитры интерфейса:
- Менять только значения переменных
- Семантика (semi-active/active) остаётся неизменной
