# Bridges — таблица мостов между окнами

Краткое описание: таблица фиксирует, какие данные переходят между окнами, и статус моста.

| From | To | Описание | Status |
|------|----|----------|--------|
| W1 (Story) | W2 (System) | Передача выбранного узла онтологии как текущего контекста System | draft |
| W2 (System) | W3 (Service) | System формирует запрос (payload) для Service на основе режима/параметров | draft |
| W3 (Service) | W1 (Story) | Результат запроса может быть отображён в Story как read‑only content | idea |

Примечание: пока статус != fixed — мост считается меняемым агентом. В описании моста указывайте конкретные поля payload и условия передачи (при рядомжениях).
