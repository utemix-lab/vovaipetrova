## Терминология

| Термин | Значение |
|--------|----------|
| **Universe** | Корневой узел графа, пользовательский мир |
| **Cryptocosm** | Скрытая механика внутри Universe |
| **Platform** (crypto-*) | Движок, схема, валидаторы — серая палитра |
| **Render** (mirror-*) | Визуализация — индиго палитра |
| **Cabins** (cabin-*) | Рабочие пространства персонажей |
| **MeaningEngine** | Универсальный движок (не знает конкретный мир) |
| **World** | Конкретный мир (VovaIPetrova — первый плагин) |
| **Track** | Ракурс рассмотрения системы (не модуль) |

## Критические правила

1. **Graph = invariant, Projection = mutable**
2. **UI не определяет онтологию** (Track 4 ≠ Track 1)
3. **Core → Projection → Visitor** (никогда наоборот)
4. **LLM — консультант, не автор изменений**
5. **Рефлексия = read-only**
